Toolbox

Just a collection of random extensions I use.

Requirements

  • iOS 15.0+
  • macOS 12.0+

Installation

Swift Package Manager

In Xcode:

  1. Click Project.
  2. Click Package Dependencies.
  3. Click +.
  4. Enter package URL: https://github.com/reddavis/Toolbox.
  5. Add Toolbox to your app target.

Other libraries
